<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository:</p>

```
git clone https://github.com/hastik07/flutter-bank_ui.git
```

<p>2. Install dependencies:</p>

```
flutter pub get
```

<p>3. Run the application:</p>

```
flutter run
```
